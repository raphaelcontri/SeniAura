## 6. Assets front-end

### `assets/style.css`

Feuille de style globale (~600 lignes). Configure :
- La typographie et les couleurs Premium.
- Le design Mantine personnalisé.
- Les animations (Pulse sur bouton Aide, Bounce sur indicateurs).

### `assets/scroll.js`

Client-side callback pour le scroll smooth vers la section "Prise en main".

### `assets/Senio.png`

Logo officiel Senio affiché dans le header.
